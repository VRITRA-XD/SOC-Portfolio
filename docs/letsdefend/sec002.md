Challenge Questions & Solutions
Question 1
What command is used to prevent the command echoing in the console?

Answer:

@echo off
This command prevents each command from being displayed in the command prompt while the script executes.

Question 2
Which tool is used to download a file from a specified URL in the script?

Answer:

bitsadmin
bitsadmin is a Windows command-line utility that can download files using the Background Intelligent Transfer Service (BITS). Although originally designed for legitimate file transfers, it is frequently abused by malware.

Question 3
What is the priority set for the download operation in the script?

Write on Medium
Answer:

FOREGROUND
The script configures the BITS download job to use the FOREGROUND priority.

Question 4
Which command is used to start localization of environment changes in the script?

Answer:

setlocal
setlocal ensures that any environment variable changes made by the script remain local and do not affect the system after the script finishes.

Question 5
Which IP address is used by the malicious code?

Answer:

193.169.255.78
This IP address is used by the batch script to retrieve the malicious payload.

Question 6
What is the name of the subroutine called to extract the contents of the ZIP file?

Answer:

UnZipFile
The script invokes the UnZipFile subroutine to extract the downloaded archive.

Question 7
Which command attempts to start the executable extracted from the ZIP file?

Answer:

start "" "FW-APGKSDTPX4HOAUJJMBVDNXPOHZ.PDF.exe"
This command launches the extracted executable, which is the final stage of the infection chain.

Question 8
Which scripting language is used to extract the contents of the ZIP file?

Answer:

VBScript
The batch script leverages VBScript to automate the extraction of the ZIP archive.

Conclusion
This challenge provides a great introduction to analyzing malicious Windows batch scripts. Even though the script is relatively simple, it demonstrates several techniques commonly used by malware, including:

Downloading payloads using bitsadmin
Extracting ZIP archives with VBScript
Executing downloaded files
Using batch scripting features like setlocal and @echo off
Understanding these techniques is an essential skill for malware analysts and SOC analysts, as similar patterns appear frequently in real-world incidents.
